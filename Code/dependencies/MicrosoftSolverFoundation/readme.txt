Microsoft.Solver.Foundation.dll for x64 is alrealy included in this project (at /Assets/). 
No operation is required to install this dependency for this project.

To install the Microsoft Solver Foundation on your computer: double click microsoftsolverfoundation64.msi.

It should be located at the default loaction:
C:\Program Files (x86)\Reference Assemblies\Microsoft\Framework\.NETFramework\v4.0\Microsoft.Solver.Foundation.dll

And there will be some examples in User\Documents\Microsoft Solver Foundation
